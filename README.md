# SunnyMemories - SillyTavern
![Version](https://img.shields.io/badge/version-1.0.9-yellow)
![SillyTavern](https://img.shields.io/badge/SillyTavern-Extension-green)

Advanced extension for summarization and memory management.
> Use separate API profiles independent from the main session
___
## Features

### 🧠 Memory & Summarization
- Generates and maintains a concise summary of the conversation, tracking key events and changes over time
### 📌 Facts Extraction
- Extracts important details (for exmaple: locations, relationships, and key decisions from the chat)
### 📚 Library System
- Structured memory storage
- Merge / split entries
- Automatic categorization
- Expiration and cleanup support
### 🎯 Quests Tracking
- Main quests, Side quests, Short-term tasks
- Dynamic updates based on context

### 📅 Calendar Integration
- Tracks events and deadlines
- Custom in-app date system
- Sync with quests and story progression
