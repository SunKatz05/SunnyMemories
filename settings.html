Topics
Enable modules and tabs
How Depth works (Depth)?
Depth controls where a memory block is inserted in chat context.
How Frequency works (Frequency / F)?
Frequency controls how often entries are injected.
Customization and colors
Use Customization to adjust accent colors, glow, and visual density. Keep only the controls you need visible, so the panel stays clean and focused.
Anti-Filter mode
Enable Anti-Filter mode only when your backend aggressively censors or trims model output. If generation quality is normal, keep it disabled.
Topics
Summary generation and modes
Summary
Summary keeps a compact story retelling for context.
Dynamic
Dynamic summary compresses as the story progresses.
Static
Static summary keeps each retelling in its original form and appends new entries.
How to write a summary prompt?
There are no strict rules, but concise and clear instructions work best.
Facts extraction
Facts
Facts store RP details that are often skipped in a regular summary.
Prompt for Facts
In Facts Prompt, list what AI must track: NPCs, clothing and its condition, relationships, and more.
Split
Split lets AI divide facts by category and send separated pieces to Library.
Library entries and injection
Library
Library stores important fragments from Summary and Facts for long-term use.
Library control panel
Moon — select one or multiple entries. Then you can apply actions to selected items.
Trash — delete selected entries.
Merge — after selecting multiple entries, AI performs smart merge: both summary/fact texts are combined into one new entry. This new entry is more compact for storage and context injection, while chronology from old entries becomes clearer, making 'before → after' easier for AI to understand.
How injection works
Each entry has its own enable toggle and injection parameters, including position, depth, and frequency.
Topics
Таймлайн и квесты
ИИ сканирует чат на квесты и цели и превращает их в простой список задач.
Типы квестов
Главные Квесты и Цели
Особые задания, на которых строится РП. Самые важные по значению.
Второстепенные Цели
Сайд-квесты, могут быть важными или не очень.
Краткосрочные Задачи
Что-то не сильно значительное, что произойдет в самое ближайшее время.
События календаря
События
События — это важные сюжетные моменты, которые уже произошли или скоро произойдут. Извлечение делает их короткими. Для более подробных записей таймлайна используй парсер в настройках парсера.
Парсер
Парсер собирает события во время RP и работает чуть шире обычного извлечения. Его можно включить в настройках парсера.
Генерация событий
ИИ может генерировать подходящие события календаря на основе выбранного контекста. Ты сам контролируешь, откуда брать контекст, диапазон дат и стиль генерации.
Стиль
Это тип событий, которые ИИ будет пытаться создавать.
Mixed — смесь разных типов.
Story — сюжетные события.
Random — случайные, «живые» события.
Social — разговоры, отношения, диалоги.
Weather — погода, климат, атмосфера.
Character — события с персонажами.
World — мир, общество, государство.
Quest — задания, цели, миссии.
Плотность событий
Low — мало событий, спокойнее, реже.
Medium — нормальный баланс.
High — много событий, более насыщенный и быстрый ритм.

Простыми словами: Low = тихо и спокойно, Medium = обычная активность, High = очень насыщенная история.
Видимость
Это то, насколько событие должно быть видно в календаре и в контексте.
Mixed — ИИ сам решает, какие сделать видимыми, а какие скрытыми.
Public — событие видно заранее и может «появляться» в контексте до даты.
Hidden — событие скрыто, пока не наступит его время.
Показывать каждые N дней
Это настройка для публичных событий. Она говорит, как часто событие должно снова «напоминать» о себе до своей даты.
Например: «каждые 3 дня» — событие будет периодически всплывать в контексте; «0» — не будет повторяться.

Saving images to album
Save images from chat directly into the album. The extension stores copied files for durable local history, not temporary message links.
Folders, sorting, and search
Use folder controls to group images by scene, arc, or character. Sort by date/name and use search to quickly navigate large collections.
Diary captions and binding
Diary mode generates short in-character captions using chat context and image metadata. Folder binding links saves to the active character for faster consistent workflow.
